## 摘要
我们提出PaintCopilot，一种协同创作的神经绘画助手，它将绘画建模为一种基于画布状态演变和历史笔触序列的无界自回归艺术行为，无需目标图像。与将绘画定义为向预定义参考进行像素重建的现有神经绘画方法不同，PaintCopilot直接从学习到的艺术动态中预测未来的笔触，类似于大型语言模型如何从先前上下文延续文本序列。该框架提出了三个互补模型：基于ViT的目标预测器，从部分画布观察中推断艺术家意图；自回归的下一个笔触预测器，通过流匹配生成时间上连贯的笔触；以及基于VAE的区域采样器，按需合成语义定位的笔触序列。基于三种可微分笔刷表示（硬圆刷、笔尖刷和二维高斯刷），该系统支持四种交互工作流：优化历史、笔触补全、区域修复和动态笔刷。通过与专业艺术家的案例研究，我们展示了PaintCopilot能够实现流畅的协同创作绘画工作流，在其中艺术家和AI在整个创作过程中持续交替控制。

## Abstract
We present PaintCopilot, a co-creative neural painting assistant that models painting as an open-ended autoregressive artistic behavior conditioned on evolving canvas states and prior brushstroke history, without requiring a target image. Unlike existing neural painting methods that frame painting as pixel reconstruction toward a predefined reference, PaintCopilot predicts future strokes directly from learned artistic dynamics, analogous to how large language models continue text sequences from prior context.   The framework proposes three complementary models: a ViT-based Target Predictor that infers artist intent from partial canvas observations, an autoregressive Next Stroke Predictor that generates temporally coherent brushstrokes via flow matching, and a VAE-based Region Sampler that synthesizes semantically localized stroke sequences on demand. Built on three differentiable brush representations (Hard Round, Brush Tip, and 2D Gaussian), the system supports four interactive workflows: Optimize History, Stroke Completion, Region Inpainting, and Dynamic Brush. Through case studies with professional artists, we demonstrate that PaintCopilot enables fluid co-creative painting workflows in which artists and AI continuously alternate control throughout the creative process.

# PaintCopilot：将绘画建模为自主艺术延续

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **研究动机**：现有神经绘画方法（如Learning to Paint、Paint Transformer等）均将绘画任务定义为“像素重建”——即给定一张目标图像，通过优化或生成笔触序列来逼近该图像。这种目标驱动范式限制了创作的开放性和艺术家的自由表达，因为真实绘画过程是开放式的、时间演进的，艺术家的意图会随画布状态动态变化。
- **核心问题**：如何构建一个无需预设目标图像、能根据当前画布状态和历史笔触自主延续绘画的AI协同创作系统？
- **整体含义**：模仿人类画家“从想象中绘画”的行为，将绘画建模为一个自回归的艺术延续问题（类似语言模型续写文本），从而支持艺术家与AI之间的流畅交替控制，实现真正的协同创作。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程（文字说明）
- **核心思想**：将绘画视为条件于演化的画布状态和过去笔触历史的开放式自回归行为，不再需要参考图像。框架包含三个互补模型和三种可微分笔刷表示，支持四种交互工作流。
- **关键技术细节**：
  1. **笔刷表示**：
     - **硬圆刷（Hard Round）**：二进制磁盘纹理，参数为位置、半径、旋转角度、压力、颜色。
     - **纹理笔尖刷（Brush Tip）**：从灰度PNG纹理加载，参数同上。
     - **2D高斯刷**：各向异性高斯核，参数为位置、x/y标准差、旋转角度、颜色。所有笔刷均支持可微分渲染（alpha-over合成），通过MSE损失优化。
  2. **数据集构建**：
     - 从3000幅古典肖像画出发，利用三个原则生成模拟人类绘画顺序的笔触序列：
       - **语义分割（BiSeNet + MediaPipe Face Mesh）**：15个语义标签，由粗到细排序。
       - **法向图（DSINE）**：根据局部法向方差排序笔触，平坦区域先用大笔刷，复杂区域后用小编刷。
       - **注意力图（DINO自注意力）**：在视觉显著性区域集中放置笔触。
     - 每个区域内部按得分排序，笔刷大小线性递减，并对完整笔触参数进行可微分优化（Adam，30轮，带early stopping）。
  3. **模型架构**：
     - **目标预测器（Target Predictor）**：
       - 输入：当前画布VAE潜码（使用Stable Diffusion VAE）。
       - 架构：ViT（Vision Transformer），预测残差Δz，输出估算目标潜码 ẑ = z_c + Δz。
       - 损失：L_target = 0.8||ẑ - z*||_1 + 0.2||ẑ - z*||_2^2，z*为真实目标潜码。
     - **下一笔触预测器（Next Stroke Predictor）**：
       - 条件：画布VAE潜码经MLP压缩为条件向量c，历史50笔触序列经因果Transformer处理（通过Adaptive Layer Normalization注入c）。
       - 生成：采用流匹配（Flow Matching），学习速度场v_θ(a_t, h, t)，从高斯先验采样后通过Euler步积分到t=1得到8维笔触参数（x, y, 压力, 大小, 角度, R, G, B）。
       - 训练目标：L_FM = E[||v_θ(a_t, h, t) - u_t||_2^2]，其中u_t = a_tar - a_src，a_t = (1-t)a_src + t a_tar。
     - **区域采样器（Region Sampler）**：
       - VAE结构，编码器处理笔触序列后输出高斯后验，解码器基于潜码z和语义标签嵌入重建局部坐标下的笔触序列。
       - 损失：L_VAE = E[||x - ẑ||_2^2] + β D_KL(q(z|x) || N(0,I))，β从0线性退火。
  4. **交互工作流**：
     - **优化历史**：利用目标预测器预测意图，对所有历史笔触进行全局可微分优化。
     - **笔触补全**：目标预测器+下一笔触预测器迭代生成后续笔触（支持lasso区域约束）。
     - **区域修复**：用户绘制lasso区域并选择语义标签，区域采样器生成完整局部笔触序列。
     - **动态笔刷**：每次笔触落笔后，在局部包围盒内对该笔触进行微调优化（背景线程，不中断绘画）。
